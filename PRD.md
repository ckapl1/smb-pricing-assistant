# Product Requirements Document (PRD)
## SMB Pricing Assistant

---

## 1. Overview and Goal

The SMB Pricing Assistant is a simple AI powered tool that helps SMB owners generate data informed pricing recommendations based on costs inputs and competition. The project has progressed from the initial business pitch document to a working prototype. Please let me know if you have any trouble opening the project.

---

## 2. Core Features (Implemented)

a. Input fields for product, cost, and competitor price  
b. Generate price recommendation button  
c. Gemini powered pricing recommendation  
d. Automatic margin percentage calculation  
e. Clear conversational text  
f. Basic front end lay out

---

## 3. Core Features (In Progress)

a. Input validation and error handling  
b. More consistent formatting  
c. Cleaner sensitivity table layout  
d. More advanced pricing elasticity modeling

---

## 4. AI Specification

AI analyzes three inputs to produce a recommending selling price, calculated margin estimate, and a short explanation of the pricing logic. The AI is responsible for transforming inputs into pricing guidance. Gemini is behind everything expect the inputs.

---

## 5. Technical Architecture

*(I had CHAT generate this section)*

### a. Frontend:
i. HTML for structure and input fields  
ii. Java Script for capturing inputs, constructing prompts, calling the Gemini API, and update the UI  
iii. Inline or light CSS for layout

### b. AI Layer:
i. Direct call to Gemini 1.5 Flash  
ii. Prompt includes product name, cost, competitor price  
iii. AI generates pricing text

### c. Computation:
i. Margin is calculated in Java Script  
ii. Senesitvity values are generated by applying fixed % adjustments

---

## 6. Prompting Strategy & Iteration Log

### a. Initial Prompt: 
Given cost per unit and competitor price, recommend a selling price and explain why.
- Returned long text answers

### b. Follow-up Prompt: 
Return a recommended price, margin percent, and a low/base/high pricing table. Format cleanly
- Good response but tone was slightly out of context

### c. Follow-up Prompt: 
Explain the pricing decision in a supportive tone for small business owners, limit to three sentences.
- Cleaner and easy to understand explanations
